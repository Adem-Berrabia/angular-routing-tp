À propos

Objectif du Projet

Ce projet est un guide complet pour apprendre le système de routing d'Angular 19. Il couvre tous les concepts essentiels du routing moderne.

Fonctionnalités

  • Routes simples et paramétrées
  • Routes enfants et lazy loading
  • Guards et resolvers
  • Navigation programmatique
  • Query parameters

Technologies

Angular 19 TypeScript Standalone Components RxJS
